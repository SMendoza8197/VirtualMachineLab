Choose a payment method for the subscription and resource group you created in a prior lab.

Name your virtual machine (any name). Select the region will your data will be stored in the cloud.

<img width="1022" alt="Steps" src="https://github.com/user-attachments/assets/4903e938-8781-4a17-b8aa-923ba1263cb7" />

For your image, choose your OS (operating system) for the virtual machine you are creating. For your size, you want something with 2 VCPUs.

Create your username and password.

Very important to follow step 7 as it can cause errors if not accepted.

Step 10: create a new name for your virtual network (any). Once created on step 13, review and create and wait for validation to be passed.
